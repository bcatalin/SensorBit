### Description

The SensorBit, designed to collect and report various sensor readings over wifi.

### Sensors

- HTU21D (Temperature + Humidity)
- MCP9808 (Temperature)
- MPL3115A2 (Pressure)
- TSL2561 (Light)

### Other Features

- ESP8266
- 3.3V operation
- Mini USB power connector
- 5v power input terminal
- FTDI onboard
- Alternate FTDI programming port
